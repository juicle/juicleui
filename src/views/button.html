按钮

创建按钮的样式是简单的。所有的按钮用 btn 的类来命名。这里添加了一些设计时需要用到的按钮类。

不同的样式

基本的按钮样式

栗子

<button class="btn btn-default">基本</button>
<button class="btn btn-primary">主要</button>
<button class="btn btn-success">成功</button>
<button class="btn btn-info">信息</button>
<button class="btn btn-warning">警告</button>
<button class="btn btn-danger">危险</button>
<button class="btn btn-link">按钮超链接</button>

按钮尺寸

小按钮，大按钮，所有的按钮只要加上 btn-sm 或 btn-lg 就可以了。

栗子

<p>
 <button class="btn btn-sm btn-default">小按钮</button>
 <button class="btn btn-sm btn-primary">小按钮</button>
</p>
<p>
 <button class="btn btn-default">基本按钮</button>
 <button class="btn btn-primary">基本按钮</button>
</p>
<p>
 <button class="btn btn-lg btn-default">大按钮</button>
 <button class="btn btn-lg btn-primary">大按钮</button>
</p>

通栏按钮

通栏显示的按钮只要添加 btn-full 就可以了。

栗子

<p><button class="btn btn-full btn-lg btn-default">full width button</button></p>
<p><button class="btn btn-full btn-lg btn-primary">full width button</button></p>

轮廓按钮

简单的创建有轮廓的按钮

栗子

<button class="btn btn-default">基本</button>
<button class="btn btn-primary-outline">主要</button>
<button class="btn btn-success-outline">成功</button>
<button class="btn btn-info-outline">信息</button>
<button class="btn btn-warning-outline">警告</button>
<button class="btn btn-danger-outline">危险</button>

圆角按钮

圆角按钮添加 btn-pill 类就可以了。

栗子

<button class="btn btn-pill btn-default-gradient">基本</button>
<button class="btn btn-pill btn-primary-outline">主要</button>
<button class="btn btn-pill btn-success">成功</button>
<button class="btn btn-pill btn-info-gradient">信息</button>
<button class="btn btn-pill btn-warning-outline">警告</button>
<button class="btn btn-pill btn-danger">危险</button>

禁用按钮

禁用的按钮使用 is-disabled 类

栗子

<button class="btn btn-default is-disabled">基本</button>
<button class="btn btn-primary is-disabled">主要</button>
