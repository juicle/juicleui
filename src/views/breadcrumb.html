面包屑/Breadcrumb

    创建面包屑导航，显示用户在网站中的位置。

    基本的面包屑

    面包屑组件由并排排列的链接和链接与链接直接的间隔组成。

    class 描述
    .breadcrumb 在<ul>元素中添加这个类定义面包屑组件。使用<a>元素作为列表中的面包屑项目。
    .active 在列表项中添加这个类，对其中一个应用一个选中的状态，并且使用<span>替代<a>元素即可。

    使用一个 <span>替代<a>元素，用来禁用某个面包屑项目。

    栗子
    • Home
    • Blog
    • Category
    • Post 
    <ul class="breadcrumb">
    <li><a href="">...</a></li>
    <li><a href="">...</a></li>
    <li><span>...</span></li>
    <li class="active"><span>...</span></li>
</ul>
