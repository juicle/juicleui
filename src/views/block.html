    修饰

    使用不同的背景颜色和 padding ，添加以下类中的一个就行了。当两个连续的块拥有相同的背景色时，padding 会自动被分开。

    class 描述
    .block-default 添加默认的背景色彩，通常是白色或类似的颜色。
    .block-muted 添加亮背景色。
    .block-primary 添加表示着重的背景色。

    Padding

    为块添加较大的 padding ，只需添加一个 .block-large 类。你还可以使用效果组件中的 .padding-* 类，来设置块中 padding 。
