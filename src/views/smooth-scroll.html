平滑滚动

    用一个漂亮的缓慢滚动效果来增强网站页面内各部分之间跳转时的效果。

    用法

    栗子
    去底部
    去顶部

    参数配置

    set 描述
    offset 滑动到的位置的偏移量
    direction 滑动的方向，可取 'top' 或 'left'
    scrollTarget 只有当你想重写默认行为的时候才会用到
    beforeScroll 滑动开始前的回调函数。
    afterScroll 滑动完成后的回调函数。
    easing 缓动效果
    speed 滑动的速度
    autoCoefficent "自动" 加速的系数
