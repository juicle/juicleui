工具提示/Tooltip

    轻松创建一个漂亮的工具提示。

    用法

    栗子

     

     

    <div class="row-fluid">
	<button class="btn btn-primary" data-title="标题" data-content="内容..." data-placement="left">左(popover)</button>
        <button class="btn btn-success" data-title="标题" data-content="内容..." data-placement="top">上(popover)</button>
	<button class="btn btn-info" data-title="标题" data-content="内容..." data-placement="right">右(popover)</button>
	<button class="btn btn-warning" data-title="标题" data-content="内容..." data-placement="bottom">下(popover)</button>
</div>

    相关配置

    使用不同的背景颜色和 padding ，添加以下类中的一个就行了。当两个连续的块拥有相同的背景色时，padding 会自动被分开。

    参数 描述
    placement 默认(auto)设置出现的位置,有auto,top,right,bottom,left,top-right,top-left,bottom-right,bottom-left,auto-top,auto-right,auto-bottom,auto-left,horizontal,vertical
    container 存放的容器
    width 容器宽度
    height 容器高度
    trigger 触发事件
    style 样式
    animation 加入css3特效
    delay show: null,
    hide: 300
    延时
    async type:'GET',
    before: function(that, xhr) {},
    success: function(that, data) {}
    error: function(that, xhr, data) {}
    异步获取
    cache 缓存
    multi 是否允许同时显示
    arrow 是否显示箭头
    title 标题
    content 内容
    closeable 是否显示关闭按钮
    direction 弹出方向
    padding 容器内间距
    type 容器的填充类型(html,iframe...)
    url 如果type是html的话,此处的值是url链接
    backdrop 背景是否打开
    dismissible 是否单击或Esc键时隐藏
    autoHide 是否自动隐藏
    offsetTop 上边距
    offsetLeft 左边距
    onShow 出现时执行回调函数
    onHide 隐藏时执行回调函数
    栗子

     

     

    <div class="row-fluid">
	<button class="btn btn-primary" id="popfun">左(popover)</button>
</div>

    相关方法

        WebuiPopovers.show(selector,options); // Show popover by jQuery selector,the  options parameter is optional
    
    WebuiPopovers.hide(selector); // 通过选择器来隐藏pop
    
    WebuiPopovers.hideAll(); // 隐藏所有pop
    
    WebuiPopovers.create(selector,options);//创建jquery的选择器
    
    WebuiPopovers.isCreated(selector);
