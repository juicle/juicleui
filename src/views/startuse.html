JUICLEUI源文件

文件包含所有的HTML,CSS,JS图片和字体文件。你也可以改进JUICLEUI。在此之前，你最好了解一些基础的前端技术。

下载JUICLEUI源文件

设置你的源代码文件

你会在根目录找到你需要设置和修改的所有文件

<!DOCTYPE html>
<html>
	<head>
		<link rel="stylesheet" href="css/yourfile.css" />
		<script src="js/yourfile.js"></script>
		<title></title>
	</head>
	<body>
	</body>
</html>

Viewport标签

JUICLEUI采用了自适应网格系统，所以在 head 标签内包含了viewport标签。 

<meta name="viewport" content="width=device-width, initial-scale=1.0">

支持的浏览器

JUICLEUI不仅支持最新的浏览器还兼容拥有较大用户量的IE较低版本的浏览器。我们支持最新的桌面和手持设备显示器，老式的IE浏览器只做到兼容排版和布局，动态及美观的效果不能完全支持。我们支持如下最新版本的5种浏览器。

Chrome Firefox Internet Explorer Opera Safari
Mac OS X
Windows
Android
iOS
