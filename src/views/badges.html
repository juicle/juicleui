角标

    使用角标来显示未读的内容或者高亮的新内容。

    应用

    应用角标配合一个导航，按钮，列表组或者更多。

    栗子 

    <button class="btn btn-default">基本按钮<span class="badge">29</span></button>
<button class="btn btn-primary">主要按钮<span class="badge">7</span></button>
<button class="btn btn-success">成功按钮<span class="badge">New</span></button>
<button class="btn btn-danger">危险按钮<span class="badge">Error</span></button>

    列表组角标

    使用角标在列表组里。添加一个修饰类来改变角标的颜色。举个栗子添加 success 类和角标类。

    添加一个浮动类 right-float 使角标浮动在右边角。

    活动的角标状态 Active 基本的角标 2 主要的角标 2 成功的角标 12 信息的角标 32 警告的角标 442 危险的角标 2 
    <div class="list-group">
  <a class="list-group-element active">活动的角标状态 <span class="badge right-float">Active</span></a>
  <a class="list-group-element">基本的角标 <span class="badge right-float">2</span></a>
  <a class="list-group-element">主要的角标 <span class="badge primary right-float">2</span></a>
  <a class="list-group-element">成功的角标 <span class="badge success right-float">12</span></a>
  <a class="list-group-element">信息的角标 <span class="badge info right-float">32</span></a>
  <a class="list-group-element">警告的角标 <span class="badge warning right-float">442</span></a>
  <a class="list-group-element">危险的角标 <span class="badge danger right-float">2</span></a>
</div>
