网格系统

    创建响应式网格布局。

    创建一个响应式网格系统我们需要一个 .container12 的类。下面我们来演示下如何创建一个4单元格和一个8单元格的响应式网格布局。 

    <div class="container12">
      <div class="grid4">
        ... 
      <div/>
      <div class="grid8">
        ... 
      </div>
</div>
    创建流体网格布局。

    创建一个流体网格布局（按照百分比来定义宽度的布局方式），只需要在 .container12 所在div的子div上加上 .row-fluid 这个类。下面我们来演示下如何创建一个4单元格和一个8单元格的流体网格布局。 

    <div class="container12">
    <div class="row-fluid">
      <div class="grid4">
        ... 
      <div/>
      <div class="grid8">
        ... 
      </div>
    </div>
</div>

    自适应的代码

    我们在构架上先引用960网格系统的样式 grid.css （即桌面显示器的框架样式），然后加入 responsive.css (包含了宽屏、平板和手机的框架样式)，最后加入用户自定义的 custom-responsive.css (来实现布局和样式分离，你只需要在 custom-responsive.css 完成最后的自适应样式编写即可)。下面是自定义自适应文件里的媒体查询隔断代码。 

    @media (min-width: 1200px) {
  /* Large Desktop screens */
  .yourclass { yourcode; }
}

@media (min-width: 980px) and (max-width: 1199px) {
  /* Desktop Screens */
  .yourclass { yourcode; }
}

@media (max-width: 979px) {
  /* Tablet screens */
  .yourclass { yourcode; }
}

@media (max-width: 737px) {
  /* Mobile screens */
  .yourclass { yourcode; }
}

    流体布局中的特定类

    有时候我们希望网格之间没有外间距，又可以实现类似 .grid1 这样简单的流体布局功能，我们添加了从 .grid-width10 到 .grid-width100 的10个特定流体布局的类。这些类只有浮动和百分比宽度的属性，可以实现无外间距和最简单的布局功能。下面我们来演示下如何创建一个30%宽和一个70%宽的流体网格布局。 

    <div class="container12">
    <div class="row-fluid">
      <div class="grid-width30">
        ... 
      <div/>
      <div class="grid-width70">
        ... 
      </div>
    </div>
</div>

    盒子模型

    考虑到兼容性，我们的框架不使用 box-sizing:border-box 这种盒子模型。无论国内外，现在似乎都没有开发者领袖强力推行这样的方案。
