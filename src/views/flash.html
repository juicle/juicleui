幻灯片

    幻灯片是网站首页基本都会有的功能，下面是一个幻灯片的使用例子。使用时加载 slide.js ，及修改 slide.less 里相对应的CSS样式。

    用法

    例子 

    <div class="slidebox">
	<div class="banner">
	    <ul>
			<li>
			    <img src="../assets/images/sunset.jpg" />				
			</li>
			<li>
			    <img src="../assets/images/subway.jpg" />				
			</li>
			<li>
			    <img src="../assets/images/shop.jpg" />				
			</li>
	    </ul>
	</div>
</div>
<script>
$('.banner').unslider({
	speed: 500,             
    	delay: 3000,
	arrows:true,            
    	dots: true, 
    	autoplay:false       
    });
</script>

    参数设置

    class 描述
    speed 速度
    delay 停留时间
    complete 滑动结束执行方法
    keys 键盘启用
    dots 圆点导航
    arrows 两侧导航
    fluid 响应式
    autoplay 自动滑动
