评论/Comment

    创建评论，比如文章的评论。

    基本的评论

    评论组件由包含头像、标题和元数据的评论header以及评论主体组成。

    class 描述
    .comment 添加这个类定义评论组件。
    .comment-header 添加这个类创建评论header。
    .comment-header 添加到 <img> 元素创建评论者的头像。
    .comment-title 添加这个类到一个标题元素中，创建评论的标题。
    .comment-meta 为段落添加这个类，创建评论的元数据。
    .comment-body 为 <div> 元素添加这个类，创建评论主体。
